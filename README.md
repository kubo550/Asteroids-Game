# Asteroids Game

## Description

Simple game of asteroids. The player controls the ship and tries to avoid the asteroids.

## Controls

Arrows: move
Space: shoot

## Play now:

Game available at: [github-pages](https://kubo550.github.io/Asteroids-Game/)
